MÓDULO DE CONSULTA Y CARGA

Ejemplo de uso

Primero carga tu fuente. Luego busca un número, IMEI o DNI para ver la red ordenada.

  • Formatos admitidos: PDF, Excel, CSV y TXT.
  • Desde la pestaña Informe puedes generar un PDF similar al modelo de inteligencia.
  • El grafo muestra: número → IMEI → números relacionados → persona/DNI.
  • El símbolo de DNI ahora aparece como persona.
Carga de archivos:
Puedes subir PDF, Excel (.xlsx/.xls), CSV o TXT. Para Excel/CSV usa columnas como: imei, numero, fecha_hora, operadora, dni, nombre, tipo.
Objetivo Número IMEI DNI / Persona Titular
Carga información o realiza una búsqueda para generar el informe automático.

INTERPRETACIÓN ANALÍTICA
