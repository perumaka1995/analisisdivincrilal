MÓDULO DE CONSULTA Y CARGA

Ejemplo de uso

Primero carga tu fuente. Luego busca un número, IMEI o DNI para ver la red ordenada.

  • Formatos admitidos: PDF, Excel, CSV y TXT.
  • Desde la pestaña Informe puedes generar un PDF similar al modelo de inteligencia.
  • El grafo muestra: número → IMEI → números relacionados → persona/DNI.
  • Cada entidad usa un icono y color propio: número, IMEI, DNI, persona y operadora.
Carga de archivos:
Puedes subir PDF, Excel (.xlsx/.xls), CSV o TXT. Para Excel/CSV usa columnas como: imei, numero, fecha_hora, operadora, dni, nombre, tipo.

🚀 Guía rápida de uso

1️⃣ Cargar información

Suba PDF, Excel, CSV o reportes. El sistema identifica DNI, teléfonos, IMEI, personas y ubicaciones.

2️⃣ Crear análisis

Revise la red de relaciones: personas, números, documentos y conexiones.

3️⃣ Editar gráfico

Clic derecho sobre un elemento: editar, eliminar, cambiar color, agregar notas o crear vínculos.

4️⃣ Crear vínculos

Seleccione nodo origen y destino. Defina tipo de relación, fecha y observación.

5️⃣ Revisar prioridades

Use colores: 🔴 crítico, 🟠 importante, 🟢 confirmado.

6️⃣ Generar informes

Descargue análisis PDF y toda la información en Excel.
